#### Como iniciar um projeto do Framework usando Grunt:
---
```shell
git clone http://github.com/guilouro/boilerplate-framework-php.git . && \
git clone http://github.com/guilouro/FRAMEWORK-PHP.git FRAMEWORK/ && \
rm -r .git && \
rm -r README.md && \
rm -r FRAMEWORK/.git && \
rm -r FRAMEWORK/.gitignore && \
git init && \
git add . && \
git status && \
git commit -m "Init"
```
